# gradle-submodules
Example for configuring Java dependencies for child projects/modules at parentlevel using Gradle with Kotlin DSL

```
RootProject(in this repo: 'gradle-submodules')
|-setings.gradle.kts
|-build.gradle.kts
|-ChildProjectA
| |-src/main/java
| |-src/test/java
|-ChildProjectB
...
```
